Very Unfunny Star Meme Generator

How it Works:

Change the parameters to make your own stupid star meme!

You can't change the font, so don't ask.

Once the picture doesn't dissapoint you anymore, go ahead and take a picture. It'll last longer.

Click to have this robo-bozo draw a unique set of stars.

Parameter Description

  • points: set the maximum amount of points that can be generated
  • radius 1: set the maximum size radius 1 can be
  • radius 2: set the maximum size radius 2 can be
  • starsColor: changes the color of the stars
  • backgroundColor:changes the color of the background
